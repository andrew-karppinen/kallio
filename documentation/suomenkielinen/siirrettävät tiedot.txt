






Paketit:

readytostart:
client lähettää serverille tiedon että on valmis aloittamaan pelin
lisäksi client lähettää 5 numeroa pitkän numerosarjan jonka pitää olla oikea

esimerkki:
"readytostart:01234"



startinfo:
serveri lähettää clientille

syntaksi:
tieto pelin aloituksesta ja kartan koko y,x suunnissa ja vaadittavien pisteiden määrä

esimerkki:
"startinfo:12,10,5"
"startinfo:y,x,vaaditut pisteet"
heti tämän perään server lähettää clientille kartan


map:


tietokoneet lähettävät toisilleensa keräämäsä pisteiden määrän


viesti voi sisältää koko kartan tai ainoastaan pelaajan sijainnin ja sen ympärillä olevat kohdat

koko kartan server lähettää clientille pelin alussa, tämän jälkeen tietokoneet lähettää toisilleen vain oman pelaajan sijainnin ja tilen vasemmalta,oikealta,ylhäältä,alhaalta,
tämä siksi jos pelaaja poistaa vieressä olevan tilen

map viestissä lähetetään myös kerättyjen  pisteiden määrän

syntaksi:
"map:kerätyt pisteet:kokokartta:sijainti:kartta merkkijonona"

silloin kun ei lähetetä koko karttaa:
"2 1,3,3,3,3"
tuosssa on tilet seuraavassa järjestyksessä:
pelaaja,oikea,ala,vasen,ylä

muuten koko kartta ylhäältä alas ja vasemmalta oikealle
z



gameexit:
tieto pelin lopetuksesta,

server lähettää myös tiedon päästiinkö maaliin vai loppuuko muuten vain
client voi lähettää ainoastaan tiedon että poistui pelistä


esimerkki:
"gameexit:True"
True = kartta läpi
False = ei läpi
